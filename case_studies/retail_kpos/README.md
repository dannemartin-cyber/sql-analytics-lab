# Retail KPI Case Study

This case study uses the retail dataset in this repo to answer core business questions.

## Questions answered
- How much revenue are we generating each month?
- Which customers generate the most revenue?
- Which products and categories perform best?
- How does customer retention look over time?
- Are there any data quality issues affecting reporting?

## Files
- `kpis.sql` — core revenue and product performance queries
- `cohort_retention.sql` — customer retention analysis by cohort month

## Goal
Translate SQL work into business insight instead of treating SQL like isolated syntax exercises.
